# Gerenciador-de-Lojas

## Sobre o projeto
Desenvolvimento de um sistema para gestão de um Shopping Center. O sistema gerencia desde a alocação de espaços físicos até regras específicas para diferentes nichos de mercado (Alimentação, Vestuário, Cosméticos, etc.), lidando com estoques dinâmicos e validações de dados.

---

## Funcionalidades
- Cadastro, exclusão e edição de lojas e produtos cadastrados no sistema. 
- Possibilidade de inserir data e endereço.
- Exibição organizada das informações de itens cadastrados no sistema.
- Possibilidade de separar lojas por categorias.
- Oferece cálculo automatizado do tamanho da loja.
- Cálculo dinâmico da folha salarial por loja.

---

## Destaques técnicos
- Lógica de Datas: "Implementação de um algoritmo próprio para validação de datas, incluindo a verificação de anos bissextos para garantir a integridade dos prazos de validade dos produtos.
- Herança e Polimorfismo: "Utilização de classes especializadas (Cosmetico, Vestuario, Alimentacao, etc.) que herdam de uma classe base Loja, permitindo comportamentos específicos para cada tipo de negócio.
- Gestão de Memória com Arrays: "Gerenciamento de estoques e listagem de lojas através de arrays, com métodos otimizados para inserção, busca e remoção de itens.

---
